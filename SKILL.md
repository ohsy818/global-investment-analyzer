# 글로벌 투자 분석 스킬

## 개요

이 스킬은 투자자가 데이터 기반의 합리적인 투자 결정을 내릴 수 있도록 다음을 제공합니다:

1. **경제 지표 수집 시스템** - 글로벌 경제 지표 및 데이터 소스 가이드
2. **뉴스 및 발언 모니터링** - 주요 인사 발언 및 정치/안보 뉴스 추적
3. **매매 신호 생성** - 기술적 분석 기반 매매 타이밍
4. **투자 전략 라이브러리** - 자산별 맞춤 전략
5. **리스크 관리 프레임워크** - 손익 관리 및 포지션 사이징

## 사용 시점

다음과 같은 투자 관련 질문이나 작업에 이 스킬을 사용:

- "지금 코스피에 투자해도 될까?"
- "레버리지 ETF는 언제 사야 해?"
- "트럼프 발언이 시장에 미치는 영향은?"
- "현재 경기 사이클은 어느 단계?"
- "인버스 ETF로 헷지해야 할 시점인가?"
- "미국 주식 vs 한국 주식 어디에 투자?"
- "Fed 금리 인상이 내 포트폴리오에 미치는 영향은?"

## 워크플로우

### 1. 거시경제 환경 분석

**목적**: 큰 그림을 파악하여 투자 방향 설정

**단계**:

1. **경제 지표 수집 스크립트 실행**:
   ```bash
   python3 scripts/collect_economic_indicators.py
   ```
   - 미국, 한국, 글로벌 지표 정보 확인
   - 필요 시 API 키 획득 (FRED, 한국은행 등)

2. **주요 지표 확인**:
   - GDP 성장률
   - 고용 지표 (실업률, 비농업 고용)
   - 인플레이션 (CPI, PCE)
   - 중앙은행 금리 정책
   - PMI (제조업/서비스업)
   - VIX (변동성 지수)

3. **경제 사이클 판단**:
   - **확장기**: 공격적 투자 (주식↑, 레버리지 고려)
   - **정점**: 방어적 전환 시작
   - **침체기**: 보수적 (채권, 현금, 인버스)
   - **회복기**: 단계적 진입

**참고 문서**: `references/economic_indicators.md`

### 3. 기술적 분석 및 매매 신호

**목적**: 구체적인 진입/청산 타이밍 결정

**단계**:

1. **매매 신호 생성 스크립트 확인**:
   ```bash
   python3 scripts/generate_trading_signals.py
   ```

2. **기술적 지표 계산**:
   
   **추세 지표**:
   - 이동평균선 (MA5, MA20, MA60)
   - MACD
   - 골든크로스/데드크로스
   
   **모멘텀 지표**:
   - RSI (14일)
   - Stochastic Oscillator
   
   **변동성 지표**:
   - 볼린저 밴드
   - ATR

3. **Python 라이브러리 활용**:
   ```python
   import pandas as pd
   import yfinance as yf
   import ta  # Technical Analysis library
   
   # 데이터 다운로드
   ticker = yf.Ticker("005930.KS")  # 삼성전자
   df = ticker.history(period="1y")
   
   # 기술적 지표 계산
   df['MA20'] = df['Close'].rolling(window=20).mean()
   df['RSI'] = ta.momentum.RSIIndicator(df['Close']).rsi()
   df['MACD'] = ta.trend.MACD(df['Close']).macd()
   ```

4. **매매 신호 판단**:
   
   **매수 신호**:
   - MA5 > MA20 > MA60 (골든크로스)
   - RSI 30-50 (과매도에서 회복)
   - MACD > Signal
   - 거래량 증가
   
   **매도 신호**:
   - MA5 < MA20 (데드크로스)
   - RSI > 70 (과매수)
   - MACD < Signal
   - 거래량 감소

### 4. 자산별 투자 전략

**참고 문서**: `references/investment_strategies.md`

#### A. 국내 주식 (KOSPI/KOSDAQ)

**대형 우량주**:
- 추세 추종 전략
- 손절: -7%
- 익절: +15%
- 장기 보유 (1년+)

**중소형 성장주**:
- 모멘텀 전략
- 손절: -10%
- 익절: +20-30%
- 단기 매매 (3-6개월)

#### B. 해외 주식 (미국)

**빅테크**:
- 장기 투자 (3년+)
- 시장 조정 시 분할 매수
- MA60 지지 확인

**성장주**:
- 52주 신고가 돌파 매수
- 첫 MA20 하향 돌파 시 매도

#### C. ETF 전략

**인덱스 ETF**:
- KODEX 200, SPY, QQQ
- 매월 정액 적립식
- 10년 이상 장기

**섹터 로테이션**:
- 경기 회복기: 금융, 산업재
- 경기 확장기: 기술, 소비재
- 경기 둔화기: 에너지, 필수소비재
- 경기 침체기: 유틸리티, 헬스케어

#### D. 레버리지 ETF

**사용 조건** (모두 충족 시):
- MA5 > MA20 > MA60 (강한 상승 추세)
- MACD 강세
- VIX < 20 (낮은 변동성)
- RSI 50-65

**리스크 관리**:
- 포지션: 전체의 10-20%만
- 손절: -10% 엄수
- 보유: 1주-1개월 (단기)
- 모니터링: 매일

**상품**:
- 국내: KODEX 레버리지, TIGER 코스닥150레버리지
- 미국: TQQQ, UPRO, SOXL

**주의**: 횡보장/약세장에서 절대 사용 금지

#### E. 인버스 ETF

**사용 조건**:
- MA5 < MA20 < MA60 (하락 정렬)
- 데드크로스
- VIX > 25
- 경기침체 신호

**진입 시점**:
- 주요 지지선 붕괴
- 지정학적 리스크 급등
- 수익률곡선 역전

**리스크 관리**:
- 포지션: 5-15%
- 손절: -8%
- 익절: +15-20%
- 보유: 며칠-2주 (초단기)

**상품**:
- 국내: KODEX 인버스
- 미국: SQQQ, SPXU

**헷지 활용**: 보유 주식 보호용

### 5. 리스크 관리

**포지션 사이징**:
- 단일 종목: 5-10%
- 섹터: 20-30%
- 레버리지: 10-20%
- 인버스: 5-15%
- 현금: 최소 10-20%

**손절 규칙**:
- 일반 주식: -7%
- 우량주: -10%
- 레버리지: -10%
- 인버스: -8%

**익절 규칙**:
- 보수적: +10-15%
- 중도적: +20-30%
- 공격적: +50%+
- 분할 익절 권장

**감정 통제**:
- 사전 계획 수립
- 규칙 준수
- FOMO 경계
- 일일 손실 한도: -3%
